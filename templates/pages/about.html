This website is a proof of competency demonstration for me, Connor Pekovic, as back-end web developer. The source code is here on Github and the aim of this project is to simply demonstrates the fundamentals of securely creating, reading, updating, and deleting (C.U.R.D.) data over the web with the Django web framework. Django is a low-code, highly-secure, future ready framework that is based in Python, the most popular programming language in the world.

I plan to make this site more beautiful over time, but proving I can create secure data interfaces is a cornerstone of professional web development in the modern era.

The Django webframework is neck-and-neck with Ruby on Rails as the #1 back-end framework used by start-ups today. There is a catch with the Django web development framework, there is a massive gulf between building simple "toy-apps" that can be created and deployed quickly and wait it takes to build "production-ready". This project demonstrates how a production-ready Django application is coded.
