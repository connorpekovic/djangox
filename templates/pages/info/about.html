This project is to serve as the perfect example as to how to make a highly secure database driven web app.

We use state of the art web technology called the Python Django Web Application Framework that runs in a low-cost, highly secure container. Started learning this method independently with this book serise and took off from there. I intentionally only implement the fundamental functions of web development here (security, containerization, REST Api's, data visualization, create/read/update/delete form's) to serve as an easy to read example for myself future projects. The aim was to keep the code base small and readable. The code base will be forever available here.

Future plans for this project include further improving my source code comments, pinning the dependencies, and all around making it a great example of a 12-factor web app for myself for the future. After that, I'll be diving into some real world enterprises as a lean entrepreneur.
